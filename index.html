My Responsibilities — Infrastructure & DBA
  • Designed complete DC-DR mirrored infrastructure across ap-south-1 and ap-southeast-1
  • Installed, configured, and administered PostgreSQL 17 on all eight nodes across both regions
  • Configured streaming replication: Primary → Read Replica (DC) and Primary → DR Standby (cross-region)
  • Implemented DR Read Replica and DR Backup Server to mirror the full DC topology in Singapore
  • Built automated pg_basebackup cron script with rotation (last 3 kept) and log tracking
  • Configured Nginx reverse proxy with security headers (X-Frame-Options, CSP, XSS-Protection)
  • Set up PM2 with startup hooks for zero-downtime persistence across reboots on all web servers
  • Wrote promote_dr.sh for sub-5-minute DR promotion and full failover sequencing
  • Deployed Netdata on web and DB servers for real-time monitoring at port 19999
  • Configured pgBadger with log_min_duration_statement=1000ms for slow query log analysis
  • Managed pg_hba.conf, scram-sha-256 replication auth, and server security hardening across all nodes
